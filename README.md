# 🎮 Palavra Secreta — Stardew Valley

## 📌 Sobre o projeto
Jogo desenvolvido em **Python** no estilo *Palavra Secreta*, onde o jogador deve adivinhar o nome de um personagem de **Stardew Valley**.

Este projeto foi desenvolvido como atividade da disciplina **Algoritmos de Programação, Projetos e Computação**, com foco em lógica, manipulação de arquivos e construção de interfaces no terminal.

---

## 🎯 Objetivo
Descobrir a palavra secreta (personagem) com base em dicas, tentando **letra por letra** antes que as tentativas acabem.

---

## ⚙️ Como funciona
- Uma palavra é sorteada aleatoriamente  
- Dicas são exibidas ao jogador  
- O jogador insere **uma letra por vez**  
- O sistema informa acertos, erros e tentativas restantes  

---

## 🧩 Funcionalidades
- Seleção aleatória de palavras  
- Sistema de dicas  
- Controle de tentativas  
- Bloqueio de letras repetidas  
- Pontuação  
- Registro de partidas em arquivo `.txt`  

---

## 🏁 Regras
- Apenas **uma letra por vez**  
- Letras corretas → ganham pontos  
- Letras erradas → perdem pontos  
- O jogo termina ao acertar a palavra ou acabar as tentativas  

---

## 🛠️ Tecnologias
- Python  
- Terminal (console)  
- Arquivos `.txt`
